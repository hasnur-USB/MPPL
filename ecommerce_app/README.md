# E-Commerce Laravel (MPPL)

Aplikasi e-commerce sederhana berbasis **Laravel 13** dengan sistem autentikasi role-based (Customer & Admin).

## ✨ Fitur yang Sudah Ada
- Autentikasi lengkap (Login, Register, Role: customer & admin)
- Katalog produk dengan gambar
- Keranjang belanja (Cart)
- Checkout & Riwayat pesanan
- Admin panel:
  - CRUD Barang (produk)
  - Manajemen Order
- Responsive design dengan Tailwind CSS + Alpine.js

## 🛠 Tech Stack
- **Backend**: Laravel 13
- **Frontend**: Tailwind CSS v4 + Alpine.js + Vite
- **Auth**: Laravel Breeze
- **Database**: MySQL / PostgreSQL / SQLite
- **Storage**: Laravel Storage (public disk)
